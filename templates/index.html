EXECAI Platform API

Welcome to the EXECAI Platform API. The following endpoints are available:

  • /api/health — Health check
  • /api/knowledge/domains — List knowledge domains
  • /api/knowledge/query (POST) — Query the knowledge base
  • /api/personas/strategic-catalyst/respond (POST) — Get a response from the Strategic Catalyst persona
  • /api/personas/strategic-catalyst/profile — Get the persona profile

Use a REST client like Postman or curl to interact with the API.
